Add the gate in two commands.

Published on PyPI. Describe your agent and tolerances in agentevals.toml, pin a baseline, then gate every change — locally or in CI. 

pipx install agent-builder-evals

# describe your agent in agentevals.toml,
# then capture a known-good baseline:
agent-evals check --update-baseline

# on every change, gate against it:
agent-evals check   # exit 1 if it regressed
# also: apples-to-apples provider comparison
agent-evals run --provider anthropic \
  --tasks coding_001,company_research_001
agent-evals run --provider openai \
  --tasks coding_001,company_research_001

agent-evals compare results/run_a.json \
  results/run_b.json
